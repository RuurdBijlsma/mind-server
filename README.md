# mind-server
Client source code: https://github.com/ruurdbijlsma/mind-client
Client live app: https://ruurd.dev/mind

## Requirements
* Python3
* Pip for Python3

## Installation

```
git clone https://github.com/RuurdBijlsma/mind-server
cd mind-server
pip install -r requirements.txt
```

## Running the server
```
cd src
python main.py
```
